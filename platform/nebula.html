    Nebulamng包管理平台介绍

    包管理平台，管理的是离线包(amr)。目前离线包是承载H5应用的主要形式，里面包含了一个业务的H5页面及静态资源文件，通过工具构建并加密打包之后生成的一个压缩文件。有关前端项目结构请参考脚手架anima-boilerplate。

    界面功能介绍

    • 平台开发环境站点地址：http://nebulamng.stable.alipay.net/clients

    功能导航

    客户端

    你可以联系站点管理员 @扶瑶 来新增你的客户端app。然后在这里可以浏览你在平台注册的app，比如钱包、聚宝、口碑商家等。你可以在右侧列表查看app信息，也可以点击 编辑 进行简单信息维护。

    产品管理

    这里管理了H5产品，比如存金宝、招财宝等。通过新增产品，维护产品名称、Owner，gitlab仓库地址等基本信息。操作栏可以新建对应产品的离线包应用。

    应用管理

    这里管理了所有的离线包应用，通过产品新增的应用都可以在这里找到。包括同一个产品的不同版本管理。 编辑一条应用记录，可以在 基本信息Tab页 修改该应用的基本信息，包括代码仓库地址、应用唯一标识、应用图标、虚拟域名、入口URL地址等。 在 版本管理Tab页 可以管理这个应用的离线包版本，并上传离线包。

    查询接口

    通过NebulaSDK内置的包管理模块，当打开某个H5应用时，会发起接口请求，获取某个版本离线包的信息，该记录会包含上节应用管理里基本信息、版本信息等。

    • 开发环境接口(仅支持post)

    http://nebula.stable.alipay.net/api/app

    可以在app内封装成rpc，在Biz层实现

    • 查询参数
    参数名 可选值 默认值 说明
    必传 -
    bundleid 客户端app bundleid
    channel offical / rc 发布渠道
    client 客户端版本
    env production / prepub 环境
    platform ios / android 客户端OS
    sdk sdk 版本
    system 系统版本
    可选 -
    query 指定应用 id 或版本来查询
    existed 指定当前客户端已存在应用及版本，用于下发增量包更新
    grayRules 灰度规则

    关于参数及返回信息的详细说明请参考 Nebula客户端API接口文档
